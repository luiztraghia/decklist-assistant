# Instalação do DeckList Assistant

1. Instale o Tampermonkey.
2. Abra o link Raw do arquivo `dist/decklist-assistant.user.js`.
3. Clique em **Instalar**.
4. Atualize a página da Liga One Piece.
5. Clique no campo de cards para o programa identificá-lo.
6. Cole a lista e inicie.

## Atualizar

Use a aba **Atualizações** do programa ou abra novamente o link de instalação.

## Desinstalar

Abra o painel do Tampermonkey e remova ou desative o DeckList Assistant.
