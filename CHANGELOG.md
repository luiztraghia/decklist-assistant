# Changelog

## 6.1.0

- Interface simplificada, sem as abas Histórico e Diagnóstico.
- Configurações reduzidas aos modos de velocidade Rápido, Médio e Lento.
- Removidos histórico, restauração de sessão e abertura automática do relatório.
- Repositório configurado como `luiztraghia/decklist-assistant`.
- Textos das abas Doar e Contato atualizados.

## 6.0.0

- Reestruturação como projeto de desenvolvimento com TypeScript e Vite.
- Build reproduzível com `vite-plugin-monkey`.
- CI para validação de pull requests.
- Release automática ao publicar tags `v*`.
- Metadados para atualização nativa pelo Tampermonkey.
- Documentação completa de instalação e publicação.
- Preservação dos recursos funcionais da versão 5.1.0.
